# Weather Diary

**Автор:** Шатанова Аделина

## Описание программы
Приложение для ведения дневника погоды: добавление, просмотр, фильтрация записей с сохранением в JSON.

## Требования
- Python 3.x

## Запуск
1. Установите Python (если не установлен).
2. Клонируйте репозиторий: git clone <ссылка>
3. Запустите: python weather_diary.py

## Примеры использования
- Добавьте запись с датой 01.01.2026, температурой 5, описанием "Солнечно", осадками "Нет".
- Используйте фильтрацию по температуре > 10°C.
